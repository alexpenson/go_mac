# go_mac
Install Go software on a mac from scratch

This should install

* Leela Zero
* Lizzie and
* GoReviewPartner

in `Go` in your home directory.
```
bash <(curl -s https://raw.githubusercontent.com/alexpenson/go_mac/master/install)
```
